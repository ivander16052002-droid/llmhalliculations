# VKR Citation Hallucinations

Проект для эксперимента по измерению галлюцинаций-цитат в ответах больших языковых моделей.

## Цель

Разработать и протестировать методику выявления и количественной оценки ошибок цитирования в ответах LLM.

## Структура проекта

- `configs/` — настройки эксперимента, моделей и промптов.
- `data/benchmark/` — вопросы бенчмарка и источники их формирования.
- `data/raw_responses/` — сырые ответы моделей.
- `data/parsed_responses/` — распарсенные ответы.
- `data/annotations/` — файлы для ручной разметки.
- `data/metrics/` — итоговые таблицы метрик
- `data/reannotation/` — подвыборка и отчеты повторной проверки
- `data/annotated/` — финальные размеченные файлы
- `src/` — код пайплайна.
- `tests/` — тесты.

## Бенчмарк

Бенчмарк состоит из 50 вопросов по пяти дисциплинам:

1. История России
2. Русский язык / лингвистика
3. Русская литература / литературоведение
4. Медиаисследования / медиафилософия
5. Цифровая гуманитаристика / история интернета

Каждый вопрос адаптирован из конкретного учебного, экзаменационного или программного источника.

## Промпты

Используются три типа промптов:

- `baseline` — нейтральный;
- `provoking` — усиливает требование привести источник;
- `conservative` — разрешает отказ при невозможности указать точный источник.

## Метрики

Основные метрики:

- SVR — Source Validity Rate
- ESR — Evidence Support Rate
- LVR — Locator Validity Rate
- RR — Refusal Rate

Аналитические метрики структуры ошибок:

- FR — Fabrication Rate
- MR — Misattribution Rate
- MR_author
- MR_title
- MR_year
- MR_container
